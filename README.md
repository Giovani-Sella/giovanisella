# giovanisella.com.br

Site pessoal de Giovani Sella — jornalista, fotógrafo e produtor audiovisual.

Publicado em [giovanisella.com.br](https://giovanisella.com.br) via GitHub Pages.

## Estrutura

```
/
├── index.html            # Página inicial
├── sobre.html            # Sobre
├── blog/
│   ├── index.html        # Lista de posts
│   ├── publicacoes.html  # Publicações em veículos
│   └── externas.html     # Links externos
├── portfolio/
│   ├── index.html
│   ├── fotografia.html
│   ├── audiovisual.html
│   └── outros.html
├── portfolio-foto/
├── audiovisual/
├── curriculo/
├── outras-atividades/
├── posts/                # Templates de post (texto, foto, vídeo)
└── assets/
    ├── css/style.css
    ├── js/script.js
    └── img/
```

## Stack

HTML, CSS e JavaScript puros — sem frameworks, sem build step, sem dependências.

## Deploy

O site é servido diretamente pelo GitHub Pages. O domínio customizado está configurado no arquivo `CNAME`.
